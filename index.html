Scenarios explored

I will test some scenarios to see the impact that the property can have when applied to different parts of the page.

  1. Default style. Visit demo
  2. Nav main menu has content-visibility:auto applied to it. Visit demo
  3. Nav main menu with content-visibility:auto and content-intrinsic-size: 100vw 100vh; specified. Visit demo.
  4. Lower 3 sections have content-visibility:auto applied. Visit demo.
  5. Lower 3 sections have content-visibility:auto and contain-intrinsic-size specified. View demo.
  6. Lower 3 sections have content-visibility:hidden applied. Visit demo.
  7. Lower 3 sections have content-visibility:hidden and contain-intrinsic-size specified applied. Visit demo.
  8. Apply content-visibility:auto to hero section to see if a penalty is incurred. Don't repeat this!! Visit demo.
